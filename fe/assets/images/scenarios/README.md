# Ảnh minh họa cho Game CV Scenarios

Thư mục này chứa ảnh minh họa cho 6 tình huống Game CV.

## Các file ảnh cần có:

1. `gift.jpg` - Quà bất ngờ (cảm xúc: vui)
2. `surprise.jpg` - Bất ngờ lớn (cảm xúc: ngạc nhiên)
3. `broken_toy.jpg` - Món đồ yêu thích bị vỡ (cảm xúc: buồn)
4. `angry.jpg` - Bạn lấy đồ (cảm xúc: tức giận)
5. `thunder.jpg` - Tiếng sấm đêm (cảm xúc: sợ hãi)
6. `disgust.jpg` - Món ăn hư (cảm xúc: ghê tởm)

## Yêu cầu ảnh:

- Định dạng: JPG hoặc PNG
- Kích thước khuyến nghị: 800x600px hoặc 16:9
- Nội dung: Phù hợp với lứa tuổi trẻ em, màu sắc tươi sáng, dễ hiểu
- Kích thước file: < 500KB mỗi ảnh

## Nguồn ảnh:

Có thể sử dụng:
- Ảnh miễn phí từ Unsplash, Pexels
- Ảnh tự vẽ/thiết kế
- Ảnh placeholder tạm thời

## Placeholder tạm thời:

Nếu chưa có ảnh, game sẽ tự động ẩn phần ảnh và chỉ hiển thị text.
